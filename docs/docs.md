# Printing a text
To print a text on the single line, use <b>print</b>.<br>
```json
[
  {
    "name": "print",
    "args": ["\"Hello, world!\""]
  }
]
```
<code style="color : red">NOTE: if you put more arguments into this array, it will print your text like if you put this: [1, 2, 3], it will output you 1 2 3.</code>
